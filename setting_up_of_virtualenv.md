## Python2.7 Virtualenv Setup

##### Installation
```
$ pip install virtualenv
```

##### Creation of virtualenv:
```
$ virtualenv -p python2.7 <desired-path>
```

#### Activate the virtualenv:
```
$ source <desired-path>/bin/activate
```

### Deactivate the virtualenv:
```bash
$ deactivate
```
